# Sistem Informasi E-Arsip Manajemen Berbasis Website

Aplikasi **Sistem Informasi E-Arsip** berbasis website yang digunakan untuk mengelola dan menyimpan dokumen arsip secara digital sehingga memudahkan pencarian, pengelolaan, dan penyimpanan data arsip dalam suatu organisasi atau instansi.

## Features

* Manajemen arsip dokumen
* Upload dan penyimpanan file arsip
* Pencarian arsip
* Dashboard informasi
* Sistem login pengguna

## Tech Stack

* PHP
* CodeIgniter
* MySQL
* HTML
* CSS
* JavaScript

## Installation

Clone repository:

```bash
git clone https://github.com/MKhumaedi/Sistem-informasi-e-arsip-manajemen-berbasis-website.git
```

Masuk ke folder project:

```bash
cd Sistem-informasi-e-arsip-manajemen-berbasis-website
```

Jalankan server:

```bash
php spark serve
```

Buka di browser:

```
http://localhost:8080
```
